# 📊 HR Analytics Dashboard

🚀 An interactive Power BI dashboard designed to analyze employee attrition, workforce demographics, salary distribution, and job role insights. This project helps organizations make data-driven HR decisions through powerful visualizations and KPI tracking.

---

# 🎯 Project Objective

The main objective of this project is to:

* Analyze employee attrition trends
* Identify departments and job roles with high turnover
* Track workforce demographics and salary insights
* Generate actionable business insights using HR data
* Build an interactive and visually appealing Power BI dashboard

---

# 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel
* SQL
* Power Query
* DAX
* Data Cleaning
* Data Visualization

---

# 📂 Dataset Information

The dataset contains HR employee records including:

* Employee ID
* Department
* Job Role
* Gender
* Age
* Salary
* Education
* Years at Company
* Attrition Status

Data cleaning and preprocessing were performed before visualization and analysis.

---

# 📈 Dashboard Features

✅ KPI Cards for:

* Employee Count
* Attrition Count
* Attrition Rate
* Average Salary
* Average Age
* Average Years at Company

✅ Interactive Visualizations:

* Attrition by Education
* Attrition by Age Group
* Attrition by Salary Slab
* Attrition by Job Role
* Attrition by Years at Company
* Department-wise Analysis

✅ Filters & Slicers for better user interaction

---

# 📊 Key Insights

* Total Employees: 1470
* Total Attrition: 237
* Attrition Rate: 16.1%
* Employees aged 26–35 showed the highest attrition
* Laboratory Technicians and Sales Executives experienced higher turnover
* Lower salary slabs showed increased attrition
* Employees with fewer years at the company were more likely to leave
